# 🏯 武侠世界 RPG 游戏

一个基于文本的武侠角色扮演游戏，具有回合制战斗系统、存档功能和丰富的游戏内容。

## 🎮 功能特点

### 核心游戏系统
- ✅ **角色创建** - 自定义角色名称，创建独特的武侠角色
- ✅ **探索系统** - 在多个地点间移动，发现隐藏的秘密
- ✅ **物品系统** - 收集和使用各种物品（武器、药品、线索等）
- ✅ **任务系统** - 接受和完成各种任务，推进游戏剧情

### 增强战斗系统
- ⚔️ **回合制战斗** - 真正的回合制战斗机制，玩家和敌人交替行动
- 🛡️ **战斗选项** - 攻击、防御、使用物品、逃跑四种选择
- ✨ **战斗特效** - 丰富的视觉特效和动画效果
- 💥 **暴击系统** - 10%概率触发1.5倍伤害的暴击
- 🧠 **敌人AI** - 智能敌人行为，包括普通攻击、强力攻击和特殊技能

### 存档系统
- 💾 **保存游戏** - 随时保存游戏进度
- 📂 **加载存档** - 从保存的进度继续游戏
- 🗑️ **存档管理** - 查看、删除存档文件
- 📁 **多存档支持** - 支持多个存档文件
- 🔄 **NPC状态保存** - 保存和恢复NPC的血量状态，确保战斗进度连贯

### 用户界面增强
- 📊 **状态显示** - 美观的角色状态面板
- 📈 **HP进度条** - 直观的血量显示
- 😊 **Unicode表情** - 丰富的表情符号增强视觉效果
- 🖼️ **战斗界面** - 带边框的战斗状态显示
- ❓ **帮助系统** - 完整的命令帮助信息

## 🕹️ 游戏命令

### 基础命令
| 命令 | 描述 | 示例 |
|------|------|------|
| `go <方向>` | 向指定方向移动 | `go 东` |
| `take <物品>` | 拾取房间中的物品 | `take 草药` |
| `use <物品>` | 使用背包中的物品 | `use 草药` |
| `attack` | 攻击当前房间的NPC | `attack` |
| `talk` | 与当前房间的NPC对话 | `talk` |

### 信息查询
| 命令 | 描述 | 示例 |
|------|------|------|
| `status` | 查看角色状态 | `status` |
| `backpack` | 查看背包物品 | `backpack` |
| `task` | 查看任务列表 | `task` |
| `accept [任务]` | 接受任务或查看可接受的任务 | `accept 初来乍到` |
| `help` | 显示帮助信息 | `help` |

### 存档功能
| 命令 | 描述 | 示例 |
|------|------|------|
| `save <名称>` | 保存游戏进度 | `save 存档1` |
| `load <名称>` - 加载已保存的游戏 | `load 存档1` |

### 游戏控制
| 命令 | 描述 |
|------|------|
| `quit` | 退出游戏 |

## 📋 任务系统

### 任务接受
| 命令 | 描述 |
|------|------|
| `accept` | 查看当前可接受的任务列表 |
| `accept <任务名称>` | 接受指定名称的任务 |

### 任务类型
| 类型 | 描述 | 特点 |
|------|------|------|
| 🎯 **主线任务** | 推进游戏剧情的重要任务 | 建议优先完成 |
| 📋 **支线任务** | 提供额外奖励和内容的可选任务 | 丰富游戏体验 |

### 任务状态
| 状态 | 描述 | 颜色标识 |
|------|------|----------|
| ⏳ **未接受** | 任务可用但尚未接受 | 白色 |
| 🔄 **进行中** | 已接受但尚未完成的任务 | 蓝色 |
| ✅ **已完成** | 任务目标已达成 | 绿色 |

## 🚀 快速开始

### 方法1：一键启动（推荐）
双击运行以下任意一个批处理文件：

| 启动文件 | 特点 | 适用场景 |
|----------|------|----------|
| `StartGame.bat` | 🚀 **游戏直接启动** | 推荐！最稳定 |
| `LaunchGame.bat` | 🎮 **带启动菜单** - 包含新游戏、加载存档等选项 | 适合需要管理存档的玩家 |
| `CompileAll.bat` | 🔧 **编译所有Java文件** | 开发者使用，重新编译项目 |

### 方法2：命令行启动
```bash
# 直接启动游戏
java -cp . com.mud.game.MudGame

# 使用游戏启动器（含菜单）
java -cp . com.mud.game.system.GameLauncher
```

### 方法3：编译运行（开发者）
```bash
# 编译所有Java文件
javac -cp . com\mud\game\*.java com\mud\game\entity\*.java com\mud\game\system\*.java com\mud\game\command\*.java com\mud\game\ui\*.java

# 运行游戏
java -cp . com.mud.game.MudGame
```

## 🗺️ 游戏世界

### 可探索地点
| 地点 | 描述 | 特点 |
|------|------|------|
| 🏘️ **村口** | 游戏起始地点 | 安全区域 |
| 🏛️ **村中心** | 村庄的中心，有古井 | 信息中心 |
| 🏪 **客栈** | 可以休息和获取信息 | 恢复体力 |
| ⚒️ **铁匠铺** | 购买和升级武器 | 装备强化 |
| 🌲 **森林** | 充满危险和机遇 | 有野狼出没 |
| 🕳️ **山洞** | 隐藏着强大的敌人 | 高风险高回报 |
| ⛰️ **山顶** | 俯瞰整个村庄 | 风景优美 |

### NPC角色
| 角色 | 功能 | 位置 | 特点 |
|------|------|------|------|
| 👨‍🌾 **村民** | 提供有用的信息和建议 | 村中心 | 友善，提供新手指导 |
| 🏪 **客栈老板** | 出售恢复物品 | 客栈 | 商人，提供交易服务 |
| ⚒️ **铁匠** | 武器专家，提供装备 | 铁匠铺 | 工匠，可强化装备 |
| 🐺 **野狼** | 森林中的危险敌人 | 森林 | 野兽，凶猛且敌对 |
| 👹 **哥布林** | 山洞中的强大对手 | 山洞 | 怪物，邪恶且敌对 |
| 🧙‍♂️ **隐士** | 修行者，提供任务 | 山顶 | 神秘，任务发布者 |

### NPC系统特色
- 🗣️ **智能对话** - 根据关系值显示不同对话内容
- 💝 **关系系统** - 与NPC建立友好或敌对关系（-100到+100）
- 📋 **任务关联** - NPC可以提供和完成任务
- ⚔️ **战斗互动** - 敌对NPC会自动攻击玩家
- 🎭 **个性特征** - 每个NPC都有独特的职业和性格描述

## 🛠️ 技术特性

### 代码架构
```
com.mud.game/
├── entity/          # 游戏实体（玩家、NPC、物品、任务、装备等）
├── system/          # 核心系统（战斗引擎、地图管理、任务管理、存档系统等）
├── command/         # 命令处理（所有游戏命令的实现）
├── ui/              # 游戏界面（战斗动画、增强UI等）
└── MudGame.java     # 主程序入口
```

### 设计模式
- 🔧 **命令模式** - 处理玩家输入的各种命令
- 📊 **状态模式** - 管理游戏和任务状态
- 💾 **序列化** - 游戏存档和加载功能

### 技术规格
- **语言**: Java 11+
- **平台**: Windows / Linux / macOS
- **编码**: UTF-8（支持中文）
- **依赖**: 纯Java，无外部依赖
